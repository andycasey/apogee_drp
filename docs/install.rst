Installation
************


Installing apogee_drp
=====================

It's easiest to pip install the software from GitHub.

.. code-block:: bash

    pip install git+https://github.com/sdss/apogee_drp.git


Dependencies
============

Here are the required packages.  Most of them can be pip installed.

- numpy
- scipy
- astropy
- dlnpyutils
- doppler
- AnniesLasso
